# SportsMate- A full-stack web application designed to connect players, coaches, and sports enthusiasts on a single platform.

## Features
- User authentication (Login/Register)
- Role-based access (User/Admin)
- Event creation and participation
- Chat and interaction system

## Tech Stack
- Frontend: HTML, CSS, Tailwind CSS
- Backend: PHP
- Database: MySQL

## How to Run
1. Install XAMPP
2. Move project to htdocs
3. Import database.sql in phpMyAdmin
4. Run localhost/SportsMate
